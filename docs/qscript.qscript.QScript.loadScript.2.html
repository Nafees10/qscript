QScript.loadScript

qscriptqscriptQScript

compiles and loads a script for execution, while keeping the previously loaded script's or byte code's functions

script is the script to load, with each line as a separate element of the array, without the trailing newline character. overWrite if true, in case of functions with matching names, the ones from previously loaded one will be replaced with new one
